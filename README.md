# GorillaOS-Camera

Distribution mirror for the GorillaOS Camera activator DLL. The launcher fetches from here when a user installs the camera through the **Camera** card on the dashboard.

## How to install

Don't download the DLL from this repo directly — use the launcher. It handles BepInEx, prerequisites, and updates.

→ [GorillaOS-Launcher](https://github.com/SuperCatCraze/GorillaOS-Launcher/releases/latest)

The camera is a paid feature available after you've purchased the appropriate tier through the website. Links to the website and Discord are inside the launcher.

## What's in here

- `GorillaOS.Camera.dll` — the latest published build
- `version.txt` — single-line version string the launcher polls
